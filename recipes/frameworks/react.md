# React Setup

## When to Use This Recipe

Use this recipe when you need to:
- Set up a React project with vibe conventions
- Configure React-specific tooling
- Understand React best practices for this boilerplate

## Current Status

This is a stub recipe. Extend it based on your project needs.

## Quick Start

```bash
# Create React app with Vite
npm create vite@latest my-app -- --template react-ts

# Or with Next.js
npx create-next-app@latest my-app --typescript
```

## Extension Points

- Add component library setup (shadcn/ui, Radix)
- Configure state management (Zustand, Jotai)
- Set up form handling (React Hook Form)
- Add animation libraries (Framer Motion)
- Configure testing (Vitest, Testing Library)
